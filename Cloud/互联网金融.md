# 互联网金融

为P2P、小贷、典当、担保、众筹等小微金融企业提供名定制化的云计算服务。

## 互联网金融方案

适合初创互联网金融公司快速搭建平台，满足在开业筹备期间快速完成各类监管验收所需系统搭建和业务功能。

架构说明：

- 初创互金公司关键系统是风控系统和网申系统，我们有大量的部署参考案例。
- 通过合理的网络和部署规划，在云平台上实现高安全和高可用的规划、方案。
- 确立面向互联网业务的应用体系，决定互联网应用在应用架构内的位置和边界。

- 提供互金业务系统架构参考

**技术架构**

- 便捷的架构部署
- 云产品弹性升级扩容
- 完善的同城高可用设计
- 强大抗DDOS攻击能力
